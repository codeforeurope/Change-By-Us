{{ d.config.site.name }} is een nieuwe online ontmoetingsplek in Amsterdam. Zoals de naam al zegt kan men op {{ d.config.site.name }} ideeën plaatsen en die in een werkbaar project omzetten in simpele stappen.

Iedereen heeft wel eens een goed idee! Wij helpen je jouw idee om te zetten in een project. Wil je inschatten of mensen jou idee tof vinden? Heb je medestanders nodig die je supporten? Wil je een oproep doen aan vrijwilligers die jou idee kunnen helpen uitvoeren? Heb je materiaal of middelen nodig? Denk je dat jou idee goed aansluit bij bepaalde organisaties? Idee voor je buurt helpt jou antwoord te krijgen op al deze vragen!

Bij {{ d.config.site.name }} kom je niet alleen halen, maar de website daagt je ook uit om na te denken over wat jij te bieden hebt. De website streeft ernaar om snel en gemakkelijk inzicht te geven in de netwerken en activiteiten in een buurt. Daarom is {{ d.config.site.name }} niet alleen een online platform, maar een plek waar je bijdrage kan leveren in het verbeteren van je leefomgeving.

{{ d.config.site.name }} vertrekt vanuit het principe van co-partnerschap en co-creatie. Stadsbewoners, initiatiefrijke stadsmakers, ondernemers, welzijnorganisaties, woningcorporaties en de overheid dragen allemaal een steentje bij aan de stad en dat wil {{ d.config.site.name }} graag versterken. Ideevoorjebuurt koppelt een online platform aan het echte leven in de buurt. Daarom is het belangrijk dat lokale organisaties zich aansluiten en hun Hulp aanbieden. De combinatie van online en offline hulp is de kracht van {{ d.config.site.name }}.

{{ d.config.site.name }} is begonnen in de zomer van 2014 als een pilotproject van Code for Europe. Code for Europe is een Europees programma bedoeld om technische oplossingen te ontwikkelen voor maatschappelijke vraagstukken in Europese steden. Code for Europe werkt met korte ontwikkeltrajecten (‘scrums’) van drie maanden en levert hoogwaardige producten op, in dit geval Idee voor je buurt. Het product wordt volledig Open Source opgezet. Deze website bestaat bijvoorbeeld ook in New York en is daar bekend als Change by Us NYC, waar het groot succes is. Het voordeel van Open Source is dat andere steden van Europa gebruik kunnen maken van deze website om met weinig inspanning een lokale versie op te zetten. Klik hier voor info over Code for Europe.

Lees meer over het gebruik van {{ d.config.site.name }} onder de Gebruikersvoorwaarden en het Privacybeleid. Bekijk ook de Veelgestelde vragen om beter te begrijpen hoe de website werkt.
